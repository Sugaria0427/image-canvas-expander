## 🇬🇧 English

### ✨ Why Canvas Expander?

Most tools only allow you to scale or crop an image **within its original canvas**, but they can't truly **change the canvas size**. Canvas Expander fills this gap:

- 🖱️ **Drag to resize the canvas** – freely drag edges or corners, just like resizing a window.
- 📐 **Any aspect ratio & position** – place the original image anywhere on the new canvas (left, right, top, bottom, or corner expansion).
- 🔍 **Mouse wheel zoom + rule‑of‑thirds guides** – fine‑tune your composition.
- 📦 **Batch processing & transparency preserved** – handle multiple images at once; PNG alpha channel is perfectly retained.

### 🚀 Live Demo

👉 [Try it now (GitHub Pages)](https://Sugaria0427.github.io/image-canvas-expander)

### 💻 Run Locally

Download this repository and open `index.html` in your browser. No dependencies, no build steps.

### 🧩 Features

- Drag to move / resize the canvas bounding box
- Preset aspect ratios (1:1, 4:3, 16:9, etc.)
- Mouse wheel zoom, double‑click to reset view
- Horizontal & vertical rule‑of‑thirds guides (9‑grid)
- Batch import, delete single images, or clear all
- Export as ZIP, preserve original format (PNG transparency / JPEG / WebP)

### 🛠️ Tech Stack

Vanilla HTML5, CSS3, and JavaScript – no frameworks.

## 🇨🇳 中文

### ✨ 为什么你需要它？

市面上多数工具只能在原图尺寸内放大、缩小或裁剪，而无法真正**修改画布大小**。Canvas Expander 解决了这个问题：

- 🖱️ **自由拖拽选框** – 像调整窗口一样随意拖动边缘/角落，直接实现画布的自由扩展。
- 📐 **任意比例与定位** – 可将原图置于新画布的任意位置（单边扩展、双边扩展均可）。
- 🔍 **滚轮缩放 + 三等分参考线** – 精细调整构图。
- 📦 **批量处理 + 保留透明** – 一次处理多张图片，PNG 透明通道完美保留。

### 🧩 核心功能

- 自由拖拽画布选框（移动 / 缩放）
- 预设比例锁定（1:1, 4:3, 16:9 等）
- 鼠标滚轮缩放视图，双击重置
- 横向/纵向三等分参考线（九宫格）
- 批量导入图片，单张删除或一键清空
- 导出为 ZIP 压缩包，保留原始格式（PNG 透明 / JPEG / WebP）
